# Coinova 🚀

A cryptocurrency tracking API built with Node.js and Express.

Coinova provides cryptocurrency market data, user authentication, watchlists, and portfolio tracking features.

## Features

- User registration and login
- JWT authentication
- Password reset functionality
- Cryptocurrency price tracking
- Search cryptocurrencies
- View top gaining coins
- Add and remove coins from watchlist
- Track portfolio holdings

## Technologies Used

- Node.js
- Express.js
- JavaScript
- MongoDB/Mongoose (planned integration)
- JSON data storage
- JWT Authentication
- bcryptjs
- External cryptocurrency APIs
